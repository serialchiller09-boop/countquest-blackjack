What are index plays?

Basic strategy assumes an average deck. When the true count is high (or low), the deck has extra tens or lows — so the mathematically best play changes. Each index tells you the true count where you switch plays.

How the spread works

True count ≤ 0 → bet 1 unit (minimum). Each full +1 true count adds 1 unit until you hit the spread cap. Keep any single bet under 2% of bankroll to limit risk of ruin. Smooth ramps avoid pit heat.
